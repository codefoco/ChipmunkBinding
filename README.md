C# bindings for Chipmunk2D 7.0.3 (compatible with Xamarin.iOS/Mac/Android/.NET/.NET Core) 

Before build fetch the submodules:

	git submodule update --init --recursive


Building
---------

	msbuild ChipmunkBinding.sln
